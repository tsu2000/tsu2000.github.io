FizzBuzz Online Game

Instructions on how to play: Click on the correct button to progress. If a number is divisible by 3, click "Fizz". If a number is divisible by 5, click "Buzz". If a number is divisible by both 3 and 5, click "FizzBuzz". If a number does not meet any of these requirements, simply click the button with the corresponding number shown. One wrong move will end the game. Have fun!
